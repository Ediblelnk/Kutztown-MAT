\section{Isomorphic Binary Structures}

\subsubsection*{Definition: Binary Algebraic Structure}
A \bld{binary algebraic structure} $<S,*>$ is a set $S$ together with a binary operation $*$.

\subsubsection*{Definition: Isomorphism}
Let $<S,*> \tand <S',*'$ be binary structures. An \bld{isomorphism} of $S$ with $S'$ is a \itl{one-to-one} function $\phi: S \mapsto S'$ such that
\[
    \forall~~ x,y \in S, \qquad \phi(x*y) = \phi(x) *' \phi(y).
\]
Notation: $<S,*> \simeq <S',*'>$
