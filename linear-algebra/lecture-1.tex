\section{Brief Review}
\subsection*{Commonly Used Sets}
\begin{itemize}
  \item $\mathbb{N}$: set of \bld{natural numbers}
        \subitem could be \itl{positive} integers
        \subitem could be \itl{nonnegative} integers
  \item $\mathbb{Z}$: set of \bld{integers}
  \item $\mathbb{Q}$: set of \bld{rational numbers}
  \item $\mathbb{R}$: set of \bld{real numbers}
\end{itemize}

\subsection*{Set Building}
To denote sets too large to just list, we use \bld{set builder} notation:
\[
  \{\text{candidate}:\text{condition}\}
\]
Examples:
\begin{align*}
   & \{x~\text{is a fruit}: x~\text{is of yellow color}\}               \\
   & \{x~\text{is a human being}: x~\text{is a president of the U.S.}\} \\
   & \{x~\text{is a city}: x~\text{is a capitol of a country}\}
\end{align*}

\subsection*{Other Notations}
\begin{itemize}
  \item $\forall~$: for all
  \item $\exists~$: there exists
  \item s.t.: such that
  \item $\rightarrow \leftarrow$: contradiction
  \item WTS: want to show
\end{itemize}
